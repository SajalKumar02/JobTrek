## 🏗️ Overview

This backend is a **modular, scalable, and secure Express.js server** that supports:

- Authentication using **JWT (access & refresh tokens)** with token rotation.
- **WebSockets** for real-time chat and notifications.
- **Production-grade** patterns like Dockerization, rate limiting, logging, monitoring, and CI/CD readiness.

> Designed to be the backbone for multiple personal or side projects.

---

## ⚙️ Features

✅ Secure JWT Authentication (Access & Refresh Tokens)  
✅ Token Rotation & Revocation Mechanism  
✅ WebSockets (Socket.IO) for Real-time Communication  
✅ PostgreSQL / MongoDB Integration  
✅ Redis Caching & Message Queue Support  
✅ Scalable Microservice-friendly Structure  
✅ Rate Limiting, Helmet, and CORS Security Layers  
✅ Logging, Metrics, and Monitoring Hooks (Pino, Prometheus, Jaeger)  
✅ CI/CD Pipeline & Docker Ready
